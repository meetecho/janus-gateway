Plugin Demo: PERC-Lite Video Room

Demo details

This is basically a clone of the plain Video Room demo, but with a key difference: it forces PERC on all participants (if you don't know what this means, check the IETF PERC charter). More precisely, it implements the PERC Lite specification, meaning all participants will share the same encrypting key within the room, a lighter approach that still prevents the server (Janus in this case) to access the media, which will be encrypted.

Notice that this only works if the publishers joining the room will use a custom Chrome version, built from this repo:

https://github.com/agouaillard/perc-webrtc

If you join with a different browser, the encryption key will be ignored, and all attempts to interact with PERC-enabled browsers will fail. Apart from this, no specific control is exposed: this demo is only meant to show how PERC-encrypted streams and communication can still work through Janus.

Press the Start button above to launch the demo.
